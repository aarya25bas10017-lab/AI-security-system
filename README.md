# Cognitive Adaptive Threat Detection System

## Overview
This project is an AI-based surveillance system that detects motion and classifies threat levels into Low, Medium, and High.

## Features
- Real-time video processing
- Motion detection using OpenCV
- Threat classification
- Automatic image capture for high threats

## Technologies Used
- Python
- OpenCV
- NumPy

## How to Run
1. Open terminal
2. Navigate to project folder:
   cd ~/Documents
3. Run:
   python3 "AI security system.py"

## Output
- Live camera feed opens
- Threat level displayed on screen
- Images saved when high threat detected
